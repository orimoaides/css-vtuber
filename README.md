# CSS VTuber Studio 🎭

**画像を一切使わず、CSSだけで描画・制御するVTuberアバター制作ツール。**
パーツを微調整し、フェイストラッキングで動かし、**単体HTML**として書き出して誰でも使えます。

![models](https://img.shields.io/badge/models-11-7c8cff) ![images](https://img.shields.io/badge/images-0-2f9e6b) ![deps](https://img.shields.io/badge/build-none-444)

## 特長

- **完全CSS描画** — アバターの全パーツ（顔・目・眉・口・髪・耳・アクセサリ）はdiv＋グラデーション＋CSS変数だけ。画像ファイルゼロ。
- **11モデル** — 🙂 ミラ（人間）/ 🐻 クマ / ✴️ クロード / 🐱 ネコ / 🤖 ロボット / 🦊 キツネ / 🐼 パンダ / 🟢 スライム / 👻 おばけ / 🐰 うさぎ / 🐲 ドラゴン。すべて同じ「スロット」構造なので、1つのパネルと1つのトラッカーで全モデルを動かせます。
- **コントロールパネル** — 顔・目・眉・口・髪・チークをスライダー/カラーで微調整。髪型（ボブ/ツインテール/ポニテ/ロング/ショート）も切替。
- **テーマ配色** — 桜/海/森/夜/炎/宝石をワンクリックで一括適用。
- **アクセサリ** — メガネ・ヘッドホン・リボン・アホ毛・ニット帽・襟（服）・✨キラキラを着脱。
- **表情プリセット9種** — 通常/笑顔/驚き/怒り/悲しい/眠い/ウインク/にっこり/じと目（滑らかに遷移）。
- **喋るテスト** — マイク無しでも口の動きをプレビュー（リップシンク風）。
- **フェイストラッキング** — Webカメラ＋MediaPipe FaceMeshで頭の向き・まばたき・口・眉・視線を反映。🎯キャリブレーション対応。
- **マウス操作（パペット）** — カメラなしでもマウスで頭と口を動かせます。
- **背景シーン** — 単色/スタジオ/夜空/星空/夕焼け、そして配信向けの**クロマキー緑**・**透明**。
- **ギャラリー** — 16の作例プリセット＋保存アバターをライブプレビューで一覧。
- **共有リンク** — 現在の設定をURLに保存して共有・復元。
- **保存スロット** — 作ったアバターをブラウザ(localStorage)に保存し、ギャラリーからプレビュー付きで呼び出し・削除。
- **JSON入出力** — ギャラリーから設定を`.json`で書き出し／読み込み（バックアップ・配布用）。
- **アクセシビリティ** — `prefers-reduced-motion` に対応（アニメ軽減）。
- **単体HTML出力** — トラッキング機能まで内蔵した自己完結ファイルを書き出し。OBSのブラウザソースにそのまま使えます（透明背景対応）。

## 使い方

```bash
# このフォルダで簡易サーバを起動（カメラ利用に http/https が必要）
python3 -m http.server 4173
# → ブラウザで http://localhost:4173 を開く
```

1. 上部のドロップダウンでモデルを選択
2. 右の**コントロールパネル**でパーツを調整（🎲ランダム / ↺リセットあり）
3. **● フェイストラッキング**でカメラ起動（初回は許可が必要・正面で🎯キャリブレーション）
4. **背景**を「透明(OBS)」や「クロマキー緑」にして配信向けに
5. **⤓ HTML出力**で単体ファイルを保存 → OBSのブラウザソースに追加

## キーボードショートカット

| キー | 動作 |
|------|------|
| `1`〜`7` | モデル切替 |
| `R` | ランダム（ガチャ） |
| `G` | ギャラリー開閉 |
| `E` | HTML出力 |
| `Space` | まばたき |
| `Esc` | ギャラリーを閉じる |

（スライダーや入力欄にフォーカス中は無効）

## OBSでの利用

1. 背景を **透明(OBS)** にして **⤓ HTML出力**
2. OBSで「ソース＋」→「ブラウザ」→「ローカルファイル」で出力したHTMLを指定
3. 透明背景でアバターが合成されます（出力ファイル内の `start` ボタンでトラッキング開始）

## 仕組み（技術メモ）

- 2つのCSS変数名前空間を `.avatar` に持たせています:
  - `--s-*` … スタイル（パネルが操作）
  - `--p-*` … ポーズ（トラッキング＋アイドルが -1..1 で操作）
- 頭の3D回転は `transform: rotateX/Y/Z()` ＋ `preserve-3d`。髪は頭の子要素にして同じ平面で回し、突き抜けを回避。
- アイドルモーションはJSが `--p-*` をrAFで書き込み、トラッキングと自然にブレンド。

