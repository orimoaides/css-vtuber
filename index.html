CSS VTuber Studio へようこそ 🎭

画像を一切使わず、CSSだけで動くVTuberアバターを作れます。

  1. モデルを選ぶ — 上のドロップダウンで 人間 / クマ / クロード / ネコ / ロボット / キツネ / パンダ / スライム / おばけ / うさぎ / ドラゴン（全11種）。
  2. 微調整 — 右のコントロールパネルで目・髪・口・色・髪型・アクセサリ・テーマ配色を調整。🎲でランダム、💾で保存。
  3. 動かす — ● フェイストラッキングでカメラ連動（初回は許可＆🎯キャリブレーション）。カメラ無しでも「マウス操作」でOK。
  4. 配信 — 背景を「透明(OBS)」や「クロマキー緑」に → ⤓ HTML出力 で単体ファイルを保存し、OBSのブラウザソースに追加。

ショートカット: 1–7 モデル / R ランダム / G ギャラリー / E 出力 / Space まばたき
